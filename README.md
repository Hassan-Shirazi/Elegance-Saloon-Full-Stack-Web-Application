# 💇‍♀️ Salon Management System

A **web-based Salon Management Application** built using **HTML, CSS, JavaScript, PHP, and MySQL**.  
This project is designed to streamline appointment scheduling, client management, staff allocation, and inventory tracking for beauty salons.

---

## 📌 Features

✅ **User Authentication** – Secure login for Admin, Receptionist, and Stylists.  
✅ **Appointment Management** – Book, reschedule, or cancel client appointments with automated notifications.  
✅ **Client Management** – Maintain client records, preferences, and appointment history.  
✅ **Inventory Management** – Track salon supplies with alerts for low stock.  
✅ **Staff Management** – Manage staff profiles, schedules, and commission tracking.  
✅ **Calendar & Scheduling** – Easy scheduling with calendar integration.  
✅ **Reports & Analytics** – Generate reports on sales, inventory, and staff performance.  
✅ **Payment & Invoicing** – Record payments and generate invoices.  
✅ **Feedback System** – Clients can submit reviews/feedback.  
✅ **Admin Panel** – Full control over users, services, and inventory.

---

## 🎯 Objectives

- Improve **operational efficiency** for salons.  
- Enhance **customer experience** with smoother booking and tracking.  
- Provide **data-driven insights** via reports and analytics.  
- Serve as a **real-life training project** for students to implement full-stack concepts.  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP (Core PHP)  
- **Database:** MySQL  
- **Tools:** XAMPP / WAMP, phpMyAdmin  
- **Version Control:** Git & GitHub
